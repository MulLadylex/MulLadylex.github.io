JavaScript对象

对象是拥有属性和方法的数据

在JavaScript中，几乎所有的事物都是对象。

对象也是一个变量，但对象可以包含多个值（多个变量），每个值以 name:value对呈现

对象定义

可以使用字符来定义和创建对象

var person = {firstName:"Lex", lastName:"L", age:21};

定义对象时可以跨越多行，空格和换行不是必须的

对象属性

键值对通常写法为name : value(键与值以冒号分割)。

属性由逗号分隔，最后一个属性后面可以没有逗号。

键值对在JavaScript对象通常称为对象属性。

访问对象属性

实例：

可以使用以下语法访问对象属性：

  • person.lastName
  • person["lastName"]

对象方法

对象的方法定义了一个函数，并作为对象的属性存储。

对象方法通过添加 () 调用 (作为一个函数)。

实例：

同时你可以访问person对象的fullName属性，他将作为一个定义函数的字符串返回：
name = person.fullName

访问对象方法

创建语法：

使用以下语句访问对象方法：

  • objectName.methodName()

在上面的实例中，fullName() 是作为 person 对象的一个方法， fullName 是作为一个属性。
如果使用 fullName 属性，不添加(), 它会返回函数的定义
