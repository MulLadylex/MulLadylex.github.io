JavaScript语法

字面量

一般固定值称为字面量，如：

  • 数字字面量：123
  • 字符串字面量："Hello World!"
  • 布尔字面量: true、false、null、undefined
  • 数组字面量：[1,2,3]
  • 对象字面量：{name:"Lex",age:21}
  • 函数表达式字面量： function(){}

变量

变量是存储数据值的容器，可以通过变量名访问变量的值。
JavaScript使用关键字var来声明变量，语法：var varName = value，varName是变量名，value是变量值。

一条语句可以声明多个变量，也可以横跨多行，语法：var var1 = value1, var2 = value2, var3 = value3

变量可以使用任何名称，但以下规则必须遵守：

  • 变量名称以字母、下划线或美元符号$开头
  • 变量名称不能以数字开头
  • 变量名称对大小写敏感（y和Y是不同的变量）
  • 保留字（关键字）不能用作变量名
  • 变量名应使用驼峰大小写

JavaScript 变量均为对象。当您声明一个变量时，就创建了一个新的对象。
变量是一个名称，字面量是一个值。

算术操作符

操作符 描述

- 减法
* 乘法
/ 除法
% 求余
++ 递增
– 递减
<< 左移
>> 右移
>>> 无符号右移
& 按位与

~ 按位非
&& 逻辑与
|| 逻辑或
! 逻辑非
== 等于
=== 全等
!= 不等于
!== 不全等
> 大于
< 小于
>= 大于等于
<= 小于等于

+运算符也可用于字符串相加：text = "Good" + "morning";。

对于字符串和数字的相加，JavaScript会把数字转换为字符串：text = "5" + 2 + 3;，结果为523。

注释

单行注释：//
多行注释：/* */

数据类型

JavaScript变量可以保存两种类型的值：基本类型值和引用类型值。JavaScript有多种数据类型：数字、字符串、数组、对象等等。

将在后面的章节中详细讲解数据类型。

函数

函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。

引用一个函数=调用函数 => 执行函数内的代码。

将在后面的章节中详细讲解函数。

字母大小写

JavaScript对大小写敏感，关键词、变量、函数名、对象名均区分大小写。

JavaScript字符集

JavaScript使用Unicode字符集。意味着JavaScript支持多种语言，如中文、日文、韩文等等。
