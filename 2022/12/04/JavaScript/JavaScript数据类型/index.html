数据类型

JavaScript拥有动态类型，意味着相同的变量可用作不同的类型。

变量的数据类型可以使用typeof操作符来获取，语法：typeof variable，variable是变量名。

值类型

值类型也是基本类型，保存在栈内存中，占用空间小，保存的是值本身，可以直接访问。

值类型包括：字符串、数字、布尔、null、undefined、Symbol

字符串

字符串是存储字符（文本）的变量，字符串可以是引号中的任意文本，可以使用单引号或双引号。

语法：var str = "Hello World!"，str是变量名，”Hello World!”是字符串值。

你可以使用索引位置来访问字符串中的每个字符，索引从0开始。

字符串可以是变量，也可以是常量，字符串常量是不可改变的，一旦创建，无法改变；字符串变量是可变的，可以通过重新赋值来改变字符串的值；同时可以使用new关键字将字符串定义为一个对象：

字符串内置属性，针对字符串对象；原始值字符串没有属性和方法。

  • constructor返回创建字符串属性的函数
  • length返回字符串的长度
  • prototype向对象添加属性和方法
  • str.charAt(index)返回指定索引位置的字符
  • str.charCodeAt(index)返回指定索引位置字符的Unicode编码
  • str.concat(string1,string2,...,stringX)连接两个或多个字符串，返回连接后的字符串
  • str.indexOf(searchValue,startIndex)返回字符串中指定值第一次出现的索引值，如果找不到返回-1，startIndex是开始查找的位置，默认为0
  • str.lastIndexOf(searchValue,startIndex)从后向前搜索字符串，返回指定值的索引值，如果找不到返回-1，startIndex是开始查找的位置，默认为字符串的长度
  • str.localeCompare(compareString)用本地特定的顺序来比较两个字符串，返回一个数字表示两个字符串的比较结果
  • str.match(regexp)找到一个或多个正则表达式的匹配，返回匹配的字符串数组
  • str.replace(regexp|substr,newSubStr|function)在字符串中查找匹配的子串，然后用新的子串替换它，并返回替换后的字符串
  • str.search(regexp)检索与正则表达式相匹配的值，返回第一个匹配的索引值，如果没有找到匹配的值，返回-1
  • str.slice(startIndex,endIndex)提取字符串的片断，并在新的字符串中返回被提取的部分，startIndex是开始提取的位置，endIndex是结束提取的位置（不包含），如果省略endIndex，则提取到字符串末尾
  • str.split(separator,limit)把字符串分割为字符串数组，separator是分隔符，limit是分割次数，如果省略，则分割所有元素
  • str.substr(startIndex,length)从起始索引号提取字符串中指定数目的字符，startIndex是开始提取的位置，length是要提取的字符数，如果省略，则提取从startIndex到字符串末尾的所有字符
  • str.substring(startIndex,endIndex)提取字符串中两个指定的索引号之间的字符，startIndex是开始提取的位置，endIndex是结束提取的位置（不包含），如果省略endIndex，则提取到字符串末尾
  • str.toLocaleLowerCase()根据主机的语言环境把字符串转换为小写，返回字符串
  • str.toLocaleUpperCase()根据主机的语言环境把字符串转换为大写，返回字符串
  • str.toLowerCase()把字符串转换为小写
  • str.toString()返回字符串
  • str.toUpperCase()把字符串转换为大写
  • str.trim()去除字符串两边的空白字符
  • str.valueOf()返回某个字符串对象的原始值

模版字符串

模板字符串是一种方便的字符串语法，可以在字符串中嵌入表达式和变量。

使用反引号 作为字符串的定界符分隔，允许多行字符串，带嵌入式表达式的字符串插值。

可以包含占位符，占位符使用${expression}来表示，expression是一个JavaScript表达式,被传递给一个函数来处理（默认是转义字符串，只执行字符串替换，然后拼接到一个字符串中）。

var name = “Lex”;
var str = Hello ${name};

数字方法：

  • num.toString()把数字转换为字符串，使用指定的基数，返回字符串
  • num.toExponential()把数字转换为指数计数法，返回字符串
  • num.toFixed()把数字转换为字符串，结果的小数点后有指定位数的数字，返回字符串
  • num.toPrecision()把数字格式化为指定的长度，返回字符串
  • Number()把对象的值转换为数字
  • parseInt()解析一个字符串，并返回一个整数
  • parseFloat()解析一个字符串，并返回一个浮点数
  • isNaN()检查是否是非数字值
  • isFinite()检查是否是有限值
  • toLocaleString()把数字转换为字符串，使用本地数字格式顺序
  • valueOf()返回一个 Number 对象的原始数字值
  • Number.MAX_VALUE可表示的最大的数
  • Number.MIN_VALUE可表示的最小的数
  • Number.POSITIVE_INFINITY正无穷大
  • Number.NEGATIVE_INFINITY负无穷大
  • Number.NaN非数字值
  • Number.EPSILON可表示的最小的数
  • Number.MAX_SAFE_INTEGER可表示的最大的整数
  • Number.MIN_SAFE_INTEGER可表示的最小的整数
  • Number.isInteger()判断是否是整数
  • Number.isSafeInteger()判断是否是安全整数

布尔

布尔值只能是true或false，语法：var bool = true，bool是变量名，true是布尔值。

布尔方法：

  • Boolean()把对象的值转换为布尔值
  • valueOf()返回一个 Boolean 对象的原始布尔值
  • toString()把逻辑值转换为字符串，并返回结果
  • toLocaleString()把逻辑值转换为字符串，并返回结果
  • Boolean.MAX_VALUE可表示的最大的数
  • Boolean.MIN_VALUE可表示的最小的数
  • Boolean.POSITIVE_INFINITY正无穷大
  • Boolean.NEGATIVE_INFINITY负无穷大
  • Boolean.NaN非数字值
  • Boolean.EPSILON可表示的最小的数
  • Boolean.MAX_SAFE_INTEGER可表示的最大的整数
  • Boolean.MIN_SAFE_INTEGER可表示的最小的整数
  • Boolean.isInteger()判断是否是整数
  • Boolean.isSafeInteger()判断是否是安全整数
  • Boolean.prototypeBoolean对象的原型对象
  • Boolean.prototype.constructor返回对创建此对象的 Boolean 函数的引用
  • Boolean.prototype.toString()返回一个表示该对象的字符串
  • Boolean.prototype.valueOf()返回一个 Boolean 对象的原始布尔值
  • Boolean.prototype.toLocaleString()返回一个表示该对象的字符串
  • Boolean.prototype[@@toStringTag]返回一个表示该对象的字符串
  • Boolean.prototype[@@unscopables]返回一个表示该对象的字符串
  • Boolean.prototype.toSource()返回一个表示该对象的字符串
  • Boolean.prototype.__proto__允许您为对象添加属性和方法
  • Boolean.prototype.__defineGetter__()绑定一个函数到一个属性的 getter
  • Boolean.prototype.__defineSetter__()绑定一个函数到一个属性的 setter
  • Boolean.prototype.hasOwnProperty()返回一个布尔值，指示对象自身属性中是否具有指定的属性

null和undefined

null和undefined都表示没有值。可以通过将值设置为null来清空对象。

null和undefined的区别：

  • null是一个表示”空”的对象，转为数值时为0；undefined是一个表示”此处无定义”的原始值，转为数值时为NaN
  • null表示”没有对象”，即该处不应该有值；undefined表示”缺少值”，就是此处应该有一个值，但是还没有定义
  • null的类型是object，undefined的类型是undefined
  • null比较特殊，它是一个空对象指针，typeof null返回object
  • undefined表示”缺少值”，就是此处应该有一个值，但是还没有定义，typeof undefined返回undefined

引用类型

引用数据类型又是对象类型，保存在堆内存中，占用空间大，保存的是值的地址，需要通过地址访问。

数组

数组是存储多个值的变量，数组中的值可以是任意数据类型，语法：var arr = [1,2,3]，arr是变量名，[1,2,3]是数组值。

更多的数组知识将在后面的章节中详细讲解。

对象

对象是复杂的数据类型，对象可以拥有多个值，对象的属性由键值对的形式定义，属性与属性间逗号分隔语法：var obj = {name:"Lex",age:21}，obj是变量名，{name:”Lex”,age:21}是对象值。

更多的对象知识将在后面的章节中详细讲解。

JavaScript 变量均为对象。当您声明一个变量时，就创建了一个新的对象。
